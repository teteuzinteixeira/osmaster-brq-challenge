## 📍 Endpoints Principais

| Type | Endpoint | Descrição |
| :--- | :--- | :--- |
| PATCH | `/api/os/{id}/complete` | Completa a ordem de serviço |
| POST | `/api/os/create` | Cria uma nova ordem de serviço |
| GET | `/api/os/report/monthly/{id}` | Gera relatório com todas as ordens de serviço do mês |

> **Dica:** você pode acessar os endpoints em: `http://localhost:8080/swagger-ui.html`

## 🛠 Tecnologias e Ferramentas

*   **Java 24+** (Core do projeto)
*   **Spring Boot** (Framework principal)
*   **Maven** (Gerenciamento de dependências)
*   **Postgres** (Banco de dados)
*   **Docker** (Containerização)
*   **AWS LocalStack** (S3)
*   **WireMock** (Mock para o endpoint consulta CEP)

## 🏗 Arquitetura e Padrões

O projeto foi construído seguindo a Arquitetura Hexagonal evoluida pela equipe de desenvolvimento da Netflix:

*   **SOLID Principles:** Código desacoplado e facil de escalar.
*   **RESTful API:** Endpoints padronizados seguindo o protocolo HTTP.

## 🚀 Como Executar a Aplicação

### Pré-requisitos
*   JDK 24 ou superior.
*   Maven 4+.
*   Docker.

### Passo a Passo
1.  **Clone o repositório:** https://github.com/TeteuzinTeixeira/OsMaster-BRQ-challenge


2.  **Suba o container**
    ```bash
    docker compose up -d
    ```

3.  **Crie as tabelas no banco de dados**
    ```
    CREATE TABLE log (
        id UUID PRIMARY KEY NOT NULL,
        payload VARCHAR(1000) NOT NULL,
        response VARCHAR(1000),
        dispatch_date TIMESTAMP NOT NULL,
        response_date TIMESTAMP,
        return_code INTEGER
    );
    
    CREATE TABLE ordem_servico (
        id int8 GENERATED BY DEFAULT AS identity PRIMARY KEY NOT NULL,
        title varchar(150) NOT NULL,
        description varchar(255) NOT NULL,
        address varchar(500) NOT NULL,
        "date" date NOT NULL,
        is_completed bool NOT NULL
    );
    ```

4.  **Configure S3 no LocalStack**
    ```bash
    $env:AWS_ACCESS_KEY_ID="test"
    ```
    ```bash
    $env:AWS_SECRET_ACCESS_KEY="test"
    ```
    ```bash
    $env:AWS_DEFAULT_REGION="us-east-1"
    ```
    ```bash
    aws --endpoint-url=http://localhost:4566 s3 mb s3://osmaster-bucket make_bucket: osmaster-bucket
    ```

5.  **Subir mock do consulta CEP**
    ```bash
    curl -X POST http://localhost:8080/__admin/mappings -H "Content-Type: application/json" -d '{"request": {"method": "POST", "url": "/consulta-cep", "bodyPatterns": [{"equalToJson": "{\"cep\": \"08676270\"}"}]}, "response": {"status": 200, "jsonBody": {"Logradouro": "Rua antonio miguel", "Número": 66, "Bairro": "Jardim Lincoln", "Cidade": "Suzano", "Estado": "São Paulo"}, "headers": {"Content-Type": "application/json"}}}'
    ```
